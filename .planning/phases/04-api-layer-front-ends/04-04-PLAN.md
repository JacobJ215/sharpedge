GET /api/v1/games/{id}/analysis response:
```typescript
interface GameAnalysis {
  game_id: string
  model_prediction: { win_probability: number; confidence: string }
  ev_breakdown: { ev_percentage: number; fair_odds: number; market_odds: number }
  regime_state: string
  key_number_proximity: unknown | null
  alpha_score: number

}
```

POST /api/v1/bankroll/simulate response:
```typescript
interface MonteCarloResult {
  ruin_probability: number; p5_outcome: number; p50_outcome: number
  p95_outcome: number; max_drawdown: number; paths_simulated: number
}
```

Locked design decisions (from CONTEXT.md):
- Monte Carlo chart: Recharts AreaChart, THREE layers:
  1. Shaded band from P5 to P95 (risk range area, low opacity fill)
  2. Solid P50 line (expected path)
  3. P5 floor dashed line (ruin scenario)
- Copilot streaming: SSE via fetch().body ReadableStream — NOT EventSource (EventSource doesn't support POST)
  Use: `const reader = response.body.getReader()` then decode chunks and split on "data: "
- Prediction markets page: same AlphaBadge + RegimeChip pattern from 04-03

    `apps/web/src/components/copilot/chat-stream.tsx` — "use client". Component state: messages: [{role, content}][], inputValue, isStreaming. SSE streaming via ReadableStream (NOT EventSource — EventSource only supports GET, copilot is POST):
    ```typescript
    const sendMessage = async (text: string) => {
      setMessages(prev => [...prev, {role: 'user', content: text}])
      setIsStreaming(true)
      let assistantMsg = ''
      setMessages(prev => [...prev, {role: 'assistant', content: ''}])

      const res = await fetch(`${API_BASE}/api/v1/copilot/chat`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ message: text }),
      })
      const reader = res.body!.getReader()
      const decoder = new TextDecoder()
      while (true) {
        const { done, value } = await reader.read()
        if (done) break
        const text = decoder.decode(value)
        for (const line of text.split('\n')) {
          if (line.startsWith('data: ') && line !== 'data: [DONE]') {
            assistantMsg += line.slice(6).replace(/\\n/g, '\n')
            setMessages(prev => [...prev.slice(0,-1), {role:'assistant', content: assistantMsg}])
          }
        }
      }
      setIsStreaming(false)
    }
    ```
    Render: scrollable message list (user messages right-aligned, assistant left-aligned), text input at bottom, Send button (disabled while isStreaming), streaming indicator dot.
