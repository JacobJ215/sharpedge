OOF pattern (NO sklearn StackingClassifier — domain-separated features require manual loop):
```python
from sklearn.model_selection import cross_val_predict
from sklearn.base import clone

tscv = TimeSeriesSplit(n_splits=5)
oof_preds = np.zeros((len(y), 5))

for col_idx, (domain_name, feature_cols) in enumerate(DOMAIN_FEATURES.items()):
    X_domain = X_all[feature_cols].values
    base_model = GradientBoostingClassifier(n_estimators=50, max_depth=3)
    # cross_val_predict with time-series CV avoids leakage
    oof_preds[:, col_idx] = cross_val_predict(
        clone(base_model), X_domain, y, cv=tscv, method="predict_proba"
    )[:, 1]
    # Train FINAL base model on full training data after OOF is generated
    base_model.fit(X_domain, y)
    # Store fitted model for inference
```

Versioned save pattern (from RESEARCH.md Pattern 4):
```python
def save_model_versioned(model_bundle, name: str, models_dir: Path) -> None:
    active_path = models_dir / f"{name}_model.joblib"
    prev_path = models_dir / f"{name}_model_prev.joblib"
    if active_path.exists():
        if prev_path.exists():
            prev_path.unlink()
        active_path.rename(prev_path)
    joblib.dump(model_bundle, active_path)
```
