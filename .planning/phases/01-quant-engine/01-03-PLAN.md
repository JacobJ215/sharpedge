Walk-forward window design (from RESEARCH.md Pattern 4):
- Expanding train / fixed test windows: window N trains on all data up to period N, tests on period N+1
- Assert zero overlap: set(train_ids) & set(test_ids) must be empty for every window
- Quality badge: "low" (<2 windows), "medium" (1+ positive ROI), "high" (>=3 windows, 2+ positive ROI), "excellent" (>=4 windows, 3+ positive ROI)

    ```python
    from dataclasses import dataclass
    from typing import Literal

    @dataclass
    class WindowResult:
        window_id: int
        train_ids: list[str]
        test_ids: list[str]
        out_of_sample_win_rate: float
        out_of_sample_roi: float
        n_bets: int

    @dataclass
    class BacktestReport:
        windows: list[WindowResult]
        overall_win_rate: float
        overall_roi: float
        quality_badge: Literal["low", "medium", "high", "excellent"]

    def create_windows(
        all_ids: list,
        n_windows: int = 4,
    ) -> list[WindowResult]:
        """Partition IDs into non-overlapping expanding-train / fixed-test windows.

        Window design: IDs are sorted. Each test window is a non-overlapping slice.
        Train window = all IDs before the test slice (expanding).
        CRITICAL: assert set(train_ids) & set(test_ids) == set() for every window.
        """

    def quality_badge_from_windows(windows: list[WindowResult]) -> str:
        """Assign quality badge based on number of windows and positive-ROI windows."""
        if len(windows) < 2:
            return "low"
        n_positive = sum(1 for w in windows if w.out_of_sample_roi > 0)
        if len(windows) >= 4 and n_positive >= 3:
            return "excellent"
        elif len(windows) >= 3 and n_positive >= 2:
            return "high"
        elif n_positive >= 1:
            return "medium"
        return "low"

    class WalkForwardBacktester:
        """Run walk-forward backtest over stored BacktestResults from BacktestEngine."""
        def run(
            self,
            results: list,  # list[BacktestResult] from BacktestEngine._fetch_resolved_predictions
            n_windows: int = 4,
        ) -> BacktestReport:
            """Partition resolved results into windows and compute per-window metrics.
            Uses result.timestamp to order chronologically before partitioning.
            """
    ```

    - Divide into n_windows+1 equal chunks. Each test window = chunk[i+1], train = all preceding chunks combined.
    - For the mock version (no real predictions yet), WindowResult.out_of_sample_win_rate and out_of_sample_roi default to 0.0 and n_bets = len(test_ids)

    Implementation details for WalkForwardBacktester.run():

    - Compute overall_win_rate and overall_roi as weighted averages by n_bets
    - Call quality_badge_from_windows(windows)

    ```python
    from dataclasses import dataclass
    from sharpedge_models.ev_calculator import american_to_implied

    def calculate_clv(bet_odds: int, closing_line_odds: int) -> float:
        """Closing line value: positive means bet placed at better price than close.
        CLV = implied_prob(closing) - implied_prob(bet)
        Positive CLV: closing line moved against you (you had the sharper line)
        """
        bet_prob = american_to_implied(bet_odds)
        closing_prob = american_to_implied(closing_line_odds)
        return closing_prob - bet_prob

    @dataclass
    class CLVStats:
        """Running CLV statistics for a portfolio of bets."""
        n_bets: int
        total_clv: float
        running_average: float
        positive_clv_rate: float  # fraction of bets with positive CLV

    def aggregate_clv(clv_values: list[float]) -> CLVStats:
        """Aggregate a list of CLV values into portfolio CLV stats."""
        if not clv_values:
            return CLVStats(n_bets=0, total_clv=0.0, running_average=0.0, positive_clv_rate=0.0)
        n = len(clv_values)
        total = sum(clv_values)
        positive_rate = sum(1 for v in clv_values if v > 0) / n
        return CLVStats(
            n_bets=n,
            total_clv=total,
            running_average=total / n,
            positive_clv_rate=positive_rate,
        )
    ```
