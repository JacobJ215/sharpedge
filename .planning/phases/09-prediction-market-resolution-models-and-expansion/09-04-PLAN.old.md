    - Same pattern — duck-typed object from row fields
    - Call PMFeatureAssembler().assemble_polymarket(obj)

    Output parquet schema (same for both venues):
    columns = FEATURE_COLUMNS + ["market_id", "category", "resolved_yes"]

  <name>Task 2: train_pm_models.py — per-category binary classifier training</name>
  <files>
    scripts/train_pm_models.py
  </files>
  <behavior>
    CLI: `python scripts/train_pm_models.py [--features-dir data/processed/prediction_markets] [--out-dir data/models/pm] [--min-markets 200]`

    For each unique category in combined kalshi_features.parquet + polymarket_features.parquet:
    1. Extract rows for that category
    2. If len(rows) < min_markets: log warning and SKIP (no error, no artifact written)
    3. Split 80/20 train/test (time-ordered — sort by market_id as proxy, or use all for fixture data which is small)
    4. Train RandomForestClassifier(n_estimators=100, random_state=42) on train set FEATURE_COLUMNS → resolved_yes
    5. Get out-of-bag predictions (oob_score=True) or use cross_val_predict for Platt calibration
    6. Fit Platt scaler: LogisticRegression(C=1.0).fit(oof_probs.reshape(-1,1), y_train)
    7. Compute Brier score on test set (after Platt calibration)
    8. Compute confidence_mult via compute_confidence_mult(brier_score)
    9. Save artifact dict to data/models/pm/{category}.joblib

    Also save data/models/pm/pm_models_report.json:
    {
      "trained_at": ISO,
      "categories": {
        "economic": {"n_samples": N, "brier_score": 0.XX, "confidence_mult": 0.XX, "trained": true},
        "crypto": {"n_samples": N, "brier_score": 0.XX, ...},
        "political": {"n_samples": 0, "trained": false, "reason": "below_min_markets"},
        ...
      }
    }

    For fixture data (50 samples per venue = 100 total, 20 per category after split):
    - min_markets defaults to 200, so ALL categories will be SKIPPED with fixture data
    - Pass --min-markets 10 in tests to force training on small fixture data
    - This is intentional: fixture data is too small for production training
