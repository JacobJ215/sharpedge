class KalshiMarket:
    ticker: str

Feature vector design (8 features, fixed length):
```
[0] last_price        — mid price at capture time (0-1 probability)
[1] spread            — bid-ask spread (0-1, Kalshi) / 0.0 for Polymarket
[2] volume_24h_norm   — volume_24h / 10000.0 (normalized)
[3] open_interest_norm — open_interest / 1000.0 (Kalshi) / liquidity / 10000.0 (Polymarket)
[4] days_to_close     — (close_time - now).days if close_time else 30.0
[5] category_int      — {economic:0, crypto:1, political:2, entertainment:3, weather:4, other:5}
[6] volume_spike      — volume_24h / (volume + 1e-9), capped at 5.0
[7] price_extremity   — abs(last_price - 0.5) * 2   (0 = 50/50, 1 = near certainty)
```
