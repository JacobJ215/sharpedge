  <behavior>
    - PMResolutionPredictor.__init__(model_dir=None, assembler=None):
      - model_dir defaults to Path("data/models/pm")
      - assembler defaults to PMFeatureAssembler() (no API clients — offline feature extraction)
      - _models: dict[str, Any] = {} — lazy-loaded per category

    - PMResolutionPredictor._is_enabled() -> bool:
      - returns True only if os.environ.get("ENABLE_PM_RESOLUTION_MODEL", "").lower() in ("true", "1")

    - PMResolutionPredictor._load_model(category: str) -> Any | None:
      - if model_dir / f"{category}.joblib" does not exist: return None
      - try: return joblib.load(model_dir / f"{category}.joblib"); except: return None
      - never raises

    - PMResolutionPredictor._predict_market(market: dict, model: Any) -> float | None:
      - assembler.assemble(market) → features (np.ndarray)
      - model.predict_proba(features.reshape(1, -1))[0][1] → probability of YES (index 1 = positive class)
      - try/except → return None on any error

    - PMResolutionPredictor.build_model_probs(markets: list[dict]) -> dict[str, float]:
      - if not self._is_enabled(): return {}
      - result: dict[str, float] = {}
      - for market in markets:
          category = assembler.detect_category(market)
          if category not in self._models:
              self._models[category] = self._load_model(category)  # lazy load once
          model = self._models[category]
          if model is None: continue  # missing model → skip → scanner uses fee-adjusted
          market_id = market.get("ticker") or market.get("condition_id")
          if not market_id: continue
          prob = self._predict_market(market, model)
          if prob is not None:
              result[market_id] = prob
      - return result
