Max drawdown formula (compute from per-window ROI sequence):
```python
def compute_max_drawdown(window_rois: list[float]) -> float:
    """Compute max drawdown from a sequence of per-window ROI values.
    Treats each window as a bet period. Returns 0.0 if all windows positive.
    Uses cumulative wealth path: wealth[i] = prod(1 + roi[j] for j <= i)."""
    import numpy as np
    wealth = np.cumprod([1.0 + r for r in window_rois])
    peak = np.maximum.accumulate(wealth)
    drawdowns = (peak - wealth) / peak
    return float(drawdowns.max()) if len(drawdowns) > 0 else 0.0
```
