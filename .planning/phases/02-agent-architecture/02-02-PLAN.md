    detect_regime.py: sync node. Calls RegimeDetector().classify_regime(**state["regime_inputs"]). Returns {"regime_result": classification}. Appends quality_warnings if confidence < 0.5. Under 60 lines.

    run_models.py: sync node. Calls EVCalculator().calculate() using state["game_context"]. Also calls KeyNumberZoneDetector if spread/total is in context. Returns {"ev_result": dict}. Under 80 lines.

    calculate_ev.py: async node. Calls MonteCarloSimulator().simulate() using ev_result data. Returns {"mc_result": MonteCarloResult | None}. Appends quality_warnings if ruin_probability > 0.05. Under 60 lines.

    validate_setup.py: sync node. Builds SetupEvalResult Pydantic model (verdict: Literal["PASS","WARN","REJECT"], reasoning: str, confidence: float). Creates ChatOpenAI(model="gpt-4o-mini", temperature=0), calls with_structured_output(SetupEvalResult). Prompt includes ev_result, regime_result.regime.value, mc_result.ruin_probability. Returns {"eval_verdict": result.verdict, "eval_reasoning": result.reasoning, "retry_count": state.get("retry_count",0) + (1 if result.verdict=="WARN" else 0)}. Under 100 lines.

    compose_alpha.py: sync node. Calls AlphaComposer().compose_alpha(edge_score, regime_scale, survival_prob, confidence_mult) derived from ev_result and mc_result. Returns {"alpha": BettingAlpha}. Under 60 lines.

    size_position.py: sync node. Computes kelly_fraction from ev_result["edge"] using half-Kelly formula: kelly = (edge / (1/win_prob - 1)) * 0.5. Clamps to [0.005, 0.25]. Returns {"kelly_fraction": float}. Under 50 lines.

    generate_report.py: sync node. Formats a Discord-embed-compatible string with verdict, alpha badge, EV%, kelly %, regime, Monte Carlo summary. Returns {"report": str}. Under 100 lines.
