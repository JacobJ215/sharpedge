    Create packages/analytics/src/sharpedge_analytics/pm_edge_scanner.py.

    The model probability problem (from RESEARCH.md Pitfall 3): No PM-specific prediction model exists in Phase 3. Use fee-adjusted price as "model probability" when the caller doesn't supply a model_probs override for a given market. This is the fair price after fees — an edge exists when the fee-adjusted price differs from the raw market price enough to exceed the threshold. Document this assumption in the module docstring.

    PMEdge dataclass (must be compatible with rank_by_alpha() which uses getattr(play, 'alpha_score', None)):
    ```python
    @dataclass
    class PMEdge:
        platform: str           # "kalshi" | "polymarket"
        market_id: str
        market_title: str
        market_prob: float      # mid_price (Kalshi) or yes_price (Polymarket)
        model_prob: float       # caller-supplied or fee-adjusted fallback
        edge_pct: float         # (model_prob - market_prob) * 100
        volume_24h: float       # USD (converted for Kalshi)
        close_time: datetime | None
        alpha_score: float      # from compose_alpha()
        alpha_badge: str        # "PREMIUM" | "HIGH" | "MEDIUM" | "SPECULATIVE"
        regime: str             # PMRegimeState.value
        regime_threshold: float # adjusted threshold used for this market
    ```

    scan_pm_edges() algorithm:
    1. For each market: compute volume_24h_usd (Kalshi: volume * mid_price; Polymarket: volume_24h directly)
    2. If volume_24h_usd < volume_floor: log.debug("skipping {market.ticker}: low liquidity ({vol:.0f} USD)"); continue
    3. Get model_prob from model_probs dict by market_id; fallback = fee-adjusted price (see RESEARCH.md Pattern 3 code example)
    4. Classify regime using classify_pm_regime() with available proxies (spread as price_variance, 1.0 as spike ratio unless Polymarket volume_24h > 5 * volume_floor)
    5. regime_threshold = classification.edge_threshold_pct
    6. edge_pct = (model_prob - market_prob) * 100
    7. If edge_pct <= regime_threshold: skip (no log needed)
    8. Compose alpha: compose_alpha(edge_score=edge_pct/100, regime_scale=PM_REGIME_SCALE[regime], survival_prob=0.95, confidence_mult=1.0)
    9. Construct PMEdge and append to results

    Return results sorted by alpha_score descending.
